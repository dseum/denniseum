PORTFOLIO

Disease Data Visualization

2019 Summer This program imported an international disease database to visualize the frequency of disease in different countries. Mathematica's GUI features such as tab control were also used to organize the visualizations. This allows easy comparisons between disease frequencies in each country.

Cold Emailing Script

2020 Fall This is a Google Apps script based program (it is based on JS with Google Apps specific functions included). With a given Google spreadsheet acting as a database of potential schools who might be interested (in a single, local area) in the Robotics For All services, the program automatically goes through and automatically sends the correct email depending on the school's grade level. This required some knowledge about GApps script functions and JS for spreadsheet scripting.

Scheduling Website

2020 Fall - Present I took on this project for the newly established tutoring branch of Robotics For All, a non-profit organization focused on helping all students receive an engaging STEM experience. The scheduling software features an account system to keep track of student/tutor priviliges and a slot system that keeps track of timezones for the internationally-based users. For the front-end, I used HTML and JS, mostly relying on Bootstrap framework and design for CSS. For the back-end, I used PHP for server-side scripting and SQL for the database.

Spotify Playlist Covers

2020 Summer This was a brief project sparked by my desire to have uniform and simple Spotify playlist (icon) covers. Spotify does automatically create covers, but it is a combination of the playlist's songs' album arts and often looks disjointed. I used Photoshop to make vectors of icons and create covers with clean pastel colors. The treble clef icon was difficult to make, so I resourced it online. A sample zip of the covers is here.
